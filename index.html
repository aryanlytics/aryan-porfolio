The Story Behind the Code

I'm Muhammad Aryan, a self-taught developer from Peshawar, now based in Islamabad. I started with data analytics, went deep into data engineering — building production pipelines with Kafka, PySpark, Airflow, and BigQuery — and now I'm channeling that systems thinking into full stack web development.

I completed FSc and have been building real projects through self-directed learning for over 8 months. I learn fast, build fast, and I'm not afraid of hard problems.
